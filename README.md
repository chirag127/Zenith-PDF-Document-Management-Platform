## ZenithPDF: Privacy-First Desktop PDF Suite

ZenithPDF is a robust, client-side desktop application designed for comprehensive local PDF management. Edit, convert, merge, split, and secure your documents with unparalleled speed and absolute privacy, leveraging the power of TypeScript, Vite, and Tauri v2.

## Features

*   **PDF Editing:** Modify existing text, add annotations, shapes, and images.
*   **Conversion:** Convert PDFs to popular formats (e.g., DOCX, TXT, JPG) and vice-versa.
*   **Merging & Splitting:** Combine multiple PDFs into one or divide large documents into smaller ones.
*   **Security:** Password protection, encryption, and permissions management.
*   **Client-Side Processing:** All operations are performed locally, ensuring maximum data privacy and security.
*   **Cross-Platform:** Built with Tauri for Windows, macOS, and Linux compatibility.

---

## Technology Stack

*   **Core:** TypeScript, Vite, Tauri v2
*   **Language:** TypeScript 6.x
*   **Bundler:** Vite 7
*   **Desktop Runtime:** Tauri v2.x
*   **Linting & Formatting:** Biome
*   **Testing:** Vitest (Unit), Playwright (E2E)
*   **Architecture:** Feature-Sliced Design (FSD)
*   **State Management:** Signals

## Getting Started

### Prerequisites

*   Node.js LTS (v20+ recommended)
*   Rust (for Tauri backend)

### Installation

1.  **Clone the repository:**
    bash
    git clone https://github.com/chirag127/ZenithPDF-Desktop-Document-Manager-Tauri-App.git
    

2.  **Navigate to the project directory:**
    bash
    cd ZenithPDF-Desktop-Document-Manager-Tauri-App
    

3.  **Install dependencies:**
    bash
    uv ruff install
